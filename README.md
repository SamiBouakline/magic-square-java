# Magic Square (Java)

An interactive Java Swing implementation of a 3×3 Magic Square puzzle.

## Rules
- Fill the grid using numbers 1–9
- Each number may appear only once
- All rows, columns, and diagonals must sum to the same value

## Features
- Interactive grid
- Real-time validation
- Win detection
- Color-coded tiles

## What I Learned
- 2D grid logic using 1D arrays
- Event-driven programming
- Game state validation
- Java Swing layouts
